About This App

📌 To-Do List App

هذا الموقع اتعمل عشان يساعدك تدير مهامك اليومية بشكل بسيط وسريع. الهدف إنه يخليك منظم أكتر وتستغل وقتك بكفاءة.

✨ المميزات:

  • إضافة المهام بسهولة
  • تحديد المهام المنجزة ✔
  • حذف المهام الغير مهمة ❌
  • حفظ تلقائي (حتى بعد ما تقفل الموقع)

🛠️ التقنية المستخدمة:

HTML + CSS + JavaScript (LocalStorage)
